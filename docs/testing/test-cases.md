# 测试用例文档

## 1. 文档说明

本文件用于沉淀《心理咨询服务预约平台》的功能测试、业务规则测试、权限测试和异常边界测试用例。测试结果将作为后续问题修复、回归验证和实训文档“测试章节”的基础材料。

统一说明：

- 执行环境：本地开发环境
- 后端地址：`http://localhost:8080`
- 前端地址：`http://localhost:5173` 或打包后由后端统一访问
- 默认测试账号：
  - 管理员：`admin / 123456`
  - 咨询师：`counselor01 / 123456`
  - 普通用户：`user01 / 123456`

测试结果标记说明：

- `未执行`
- `通过`
- `不通过`

> 首轮测试执行日期：2026-06-18
> 执行人：单人开发
> 执行方式：API 接口手工测试（curl）

## 11. 当前测试执行策略

当前阶段采用以下测试推进方式：

1. 先建立测试用例基线
2. 每完成一轮代码迭代后执行相关测试
3. 发现问题后登记到缺陷记录表
4. 修复完成后写入回归测试记录

后续随着企业级方向增强项推进，本文件将继续补充参数校验、权限限制和异常边界测试。
