# 统一品牌图标设计记录

## 本轮目标

为项目补充统一的品牌图形，使浏览器标签页图标与左侧导航顶部品牌图标保持一致，提升项目完成度与识别度。

## 图标设计思路

- 图标主体使用蓝绿渐变，延续当前系统清爽、专业的后台视觉风格。
- 中心图形采用“心形 + 心理健康脉冲线”的组合，表达心理咨询、关怀服务与健康管理的业务定位。
- 结构尽量简洁，保证在浏览器标签页的小尺寸场景下仍然可识别。

## 已完成内容

1. 新增统一 SVG 图标
   - 文件：`frontend/public/brand-mark.svg`
   - 同时作为 favicon 与侧边栏品牌图标使用。

2. 浏览器标签页图标接入
   - `frontend/index.html`
   - 增加 `link rel="icon"`，引用 `/brand-mark.svg`

3. 左侧导航顶部品牌图标接入
   - `frontend/src/layout/MainLayout.vue`
   - 将原来的文字块替换为图标图片，统一使用 `brand-mark.svg`

## 结果

- 浏览器标签页与左侧导航顶部图标已统一。
- 图标与项目中文品牌文案风格保持一致。
- 不依赖外部素材平台，后续可继续基于 SVG 自定义调整颜色、圆角、细节和导出 PNG。

## 如果后续还想继续优化

可以继续做以下升级：

1. 导出多尺寸 PNG 图标
   - 适合桌面快捷方式、社交分享或部署后的多端图标适配。

2. 增加登录页品牌图标
   - 让登录页与后台主布局形成更强一致性。

3. 增加深浅两套品牌图标
   - 适合后续暗色主题或宣传图使用。
