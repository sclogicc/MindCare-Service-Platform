# 站点背景氛围升级记录

## 本轮目标

为整个前端系统增加更完整的背景氛围，让页面从普通后台样式提升到更统一、更高级、且贴合“心理咨询服务预约平台”主题的视觉层级。

## 设计方向

本轮没有直接使用重型位图背景，而是采用项目内置的 `SVG` 环境背景图，原因如下：

1. 更清晰
   - 在高分辨率屏幕下不会发糊。

2. 更轻量
   - 适合作为整站背景长期加载，不会给列表页和后台页面带来过重资源负担。

3. 更可控
   - 可以精确控制亮度、层次、透明度，避免压住表格、表单和图表内容。

## 背景元素构成

新增背景图文件：

- `frontend/public/mindcare-ambient-bg.svg`

图形语言包含：

- 蓝绿渐变光晕
- 柔和数据流曲线
- 节点连线
- 心理健康相关的脉冲意象
- 大面积留白与低对比网格

整体希望传递：

- 专业
- 平静
- 科技感
- 治愈感

## 已接入内容

1. 全局背景
   - `frontend/src/styles.css`
   - 通过 `body::before` 接入环境背景图。

2. 全局纹理层
   - `frontend/src/styles.css`
   - 通过 `body::after` 增加更轻的网格感，增强空间层次。

3. 卡片玻璃感增强
   - `frontend/src/styles.css`
   - 对 `page-card` 的透明度、模糊和阴影进行微调，让背景存在感更强但不干扰阅读。

4. 登录页背景统一
   - `frontend/src/views/login/LoginView.vue`
   - 移除登录页原有独立背景，改为复用全站统一环境背景。

## 结果

- 登录页和后台主界面处于同一视觉语言下。
- 背景不再只是纯色和简单渐变，而是具备主题化氛围。
- 表格、表单和图表仍然保持清晰可读。

## 后续可继续优化的方向

1. 单独为登录页做更强烈的品牌主视觉版本
2. 增加浅动效粒子或缓慢漂浮光斑
3. 为深色主题补一套对应背景

## 补充：多版本背景方案

为了便于后续快速切换，本轮额外保留了多套背景版本：

- `frontend/public/mindcare-ambient-bg.svg`
  - 当前默认方案，偏克制、偏高级、适合后台长期使用

- `frontend/public/mindcare-ambient-bg-serene.svg`
  - 更柔和、更治愈

- `frontend/public/mindcare-ambient-bg-tech.svg`
  - 更科技、更锐利

如果后续需要切换，只需要修改：

- `frontend/src/styles.css`
- 变量：`--mc-ambient-bg`
