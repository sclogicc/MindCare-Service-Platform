# 前端页面与 UI 说明

本文档用于汇总前端结构、页面职责和后续几轮 UI 打磨结果，替代原来按轮次拆开的零散记录。

## 1. 前端结构

前端基于：

- Vue 3
- Vite
- Vue Router
- Pinia
- Axios
- Element Plus
- ECharts

主要目录：

- `frontend/src/views`：页面
- `frontend/src/layout`：后台整体布局
- `frontend/src/router`：路由
- `frontend/src/stores`：状态管理
- `frontend/src/api`：接口封装
- `frontend/src/utils`：请求封装、鉴权、格式化工具

## 2. 页面规划

当前主要页面包括：

- 登录页
- 首页 / 仪表盘
- 咨询师管理页
- 时间段管理页
- 预约管理页
- 新增预约页
- 咨询记录页
- 反馈评价页
- 统计报表页

## 3. 后台布局设计

后台采用典型管理系统布局：

- 左侧侧边导航
- 顶部当前页面标题和用户信息
- 右侧主内容区

布局文件：

- `frontend/src/layout/MainLayout.vue`

设计目标：

- 页面切换时保持框架稳定
- 导航、内容区和顶部信息区职责清晰
- 更适合简历项目展示和后台联调

## 4. 主要 UI 打磨内容

### 4.1 侧边栏与导航

已完成：

- 统一导航宽度和层级结构
- 压缩导航高度，避免必须滚动
- 将用户信息从侧边栏移到顶部区域
- 修复导航激活态背景不稳定的问题

### 4.2 页面风格统一

已完成：

- 列表页统一卡片容器风格
- 顶部标题区统一排版
- 表格区、筛选区、表单区统一间距和边框风格
- 页面背景和组件阴影风格统一

### 4.3 表格宽度统一

已完成：

- 预约管理、反馈评价、咨询记录等页面统一表格容器策略
- 减少横向滚动条出现频率
- 保证页面在同一布局下尺寸表现接近

### 4.4 品牌与中文化

已完成：

- 页面标题统一改为中文
- 左侧导航标题改为中文
- 浏览器标签页标题改为中文
- 统一接入品牌图标 `brand-mark.svg`

### 4.5 背景与氛围

已完成：

- 为整体页面补充更完整的氛围背景
- 使用 SVG 背景而不是重型位图
- 在视觉上更贴合“心理咨询服务平台”主题

### 4.6 登录页体验修正

已完成：

- 统一登录表单输入框背景样式
- 修复登录成功时出现多余错误弹窗的问题
- 将退出登录提示与登录成功提示区分开

## 5. 当前前端特点

当前前端不追求复杂炫技，而是强调：

- 页面完整可用
- 布局统一
- 交互稳定
- 联调顺畅
- 风格更接近真实后台管理系统

## 6. 后续可继续优化的方向

- 增加表单校验提示的一致性
- 对不同角色做更明显的菜单可见性控制
- 为图表页补充时间范围筛选联动
- 增加空状态、加载态、异常态组件
- 补充移动端适配策略
