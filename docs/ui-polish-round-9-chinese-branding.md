# 中文品牌与导航文案调整

## 本轮目标

统一前端可见品牌文案与导航入口语言，移除顶部、左侧导航、标签页中的英文表达，整体改为中文后台风格。

## 已调整内容

1. 浏览器标签页
   - `frontend/index.html` 默认标题改为“心理咨询服务预约平台”。
   - `frontend/src/router/index.js` 增加基于路由 `meta.title` 的动态标题设置。

2. 左侧导航栏
   - 品牌主标题改为“心理咨询平台”。
   - 品牌副标题改为“服务预约与管理后台”。
   - 左上角图形文字由 `MC` 改为“心”。
   - 导航菜单标题与说明统一为中文表达。

3. 顶部头部区域
   - 顶部品牌提示改为“心理咨询服务预约平台”。
   - 默认页标题兜底改为“后台管理”。
   - 用户区域状态与退出按钮改为中文。

4. 页面局部英文入口
   - 登录页顶部标识改为中文。
   - 登录卡片说明改为“后台登录”。
   - 首页标识改为“首页总览”。
   - 报表页标识改为“统计中心”。

## 结果

- 浏览器标签页、左侧导航、顶部品牌区已统一中文。
- 页面入口文案更接近正式中文业务系统。
- 路由切换时标签页标题会随页面同步变化。
